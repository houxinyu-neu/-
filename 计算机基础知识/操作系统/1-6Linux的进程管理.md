# Linux的进程管理

## Linux进程的相关概念

1. 进程类型

   + 前台进程

     具有终端，可与用户交互。

   + 后台进程

     不占有终端，不和用户交互。

   + 守护进程

     + 守护`daemon`进程是特殊的后台进程。

     + 守护进程在系统引导时启动，运行到系统关闭。

       ​	如：`sshd`、`httpd`、`musqld`

2. 进程的标记

   + 进程ID

     + 进程为唯一标记（非负整数）

     + 特殊进程

       + ID = 0，为`idle`进程，是系统创建的第一个进程。

       + ID = 1，为`init`进程，是0号进程的子进程，完成系统初始化。是所有用户进程的祖先进程。
